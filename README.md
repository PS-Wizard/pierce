# pierce-v2

Lean Svelte deck for Pierce.

## Stack
- Svelte + Vite
- Bun
- Tailwind CSS v4
- Native View Transitions

## Components
- `Slides` - keyboard-driven slide shell
- `Flip` - wrapper that assigns a matching `view-transition-name`
- `Chessboard` - FEN-based board in pieces or bitboard mode
- `CodeBlock` - clean, no-highlight code surface

## Run
```bash
bun install
bun dev
```
